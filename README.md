# 🛍️ PlatziFakeStore API

Uma aplicação web que consome a API Fake Store do Platzi para exibir produtos em uma loja virtual.

## Descrição

Projeto desenvolvido para praticar consumo de APIs externas, exibindo catálogo de produtos de uma loja fictícia com interface responsiva.

## Tecnologias

- **HTML5** - Estrutura da página
- **CSS3** - Estilização e design responsivo
- **JavaScript** - Consumo da API e interatividade
- **Fake Store API (Platzi)** - Base de dados de produtos

## Funcionalidades

- ✅ Listagem de produtos da API Fake Store
- ✅ Exibição de detalhes dos produtos
- ✅ Interface responsiva para desktop e mobile
- ✅ Consumo de dados em tempo real
